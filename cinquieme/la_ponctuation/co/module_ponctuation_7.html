Contenu :

Ajoutez les virgules ou les deux-points oubliés dans les énoncés suivants.

Question

a) On peut trouver toutes sortes de fruits fraises cerises abricots pêches en cette saison.

Solution

a) Dans une énumération*, deux-points précèdent les exemples séparés par des virgules.

On peut trouver en cette saison toutes sortes de fruits : fraises, cerises, abricots, pêches.

Question

b) Pierre est à l'hôpital il a été renversé par une voiture.

Solution

b) On introduit une explication* au moyen de deux-points.

Pierre est à l'hôpital : il a été renversé par une voiture.

Question

c) Le général s'est écrié —  Silence dans les rangs  !

Solution

c) Après un verbe de déclaration*, on introduit les paroles avec deux-points.

Le général s'est écrié : —  Silence dans les rangs  !

Question

d) Étude du déplacement le bond chez le lapin.

Solution

d) On introduit un exemple avec deux points.

Étude du déplacement : le bond chez le lapin.
